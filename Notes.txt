"Design Pattern"
Design pattern are the well proved solution of commonly occurring problems in software design
Design patterns represent an ideas

Types:

1.Creational Design Patterns:
All patterns that are related to the creation of objects
Example: Factory, Singleton and Builder patterns

2. Structural Design Patterns:
All patterns that are related to structure like how we structure our code like class, objects etc
Example: Proxy, adapter design pattern

3. Behavioral Design Patterns:
It related to the object interaction like how two objects are interact with each other that it maintains its
loose coupling.
Example: Observal, State, Iterator patterns
